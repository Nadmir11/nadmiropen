# Nadmir Open Golf Tournament

Welcome to the official repository for the **Nadmir Open**, an invitational golf tournament held across multiple courses. 

## About
- **Event:** Nadmir Open
- **Founder:** Nadmir
- **Dates:** April 2026
- **Location:** Jacksonville, FL

## Site Overview
This site is built using HTML and CSS, hosted free via GitHub Pages. It includes:
- Home page with event overview
- About the founder
- Competitors and bios
- Tournament rules
- 4-day schedule
- Courses with images

## Notes
- All edits can be done directly on GitHub by editing `index.html`.
- Images should be added via the `courses` section or `img` folder if created.
